## Описание

Movie Library - это GUI-приложение для управления личной коллекцией фильмов. Приложение позволяет добавлять, просматривать, фильтровать и удалять фильмы. Данные сохраняются в формате JSON для постоянного хранения.

## Функциональность

- ✏️ Добавление фильмов с информацией о названии, жанре, годе выпуска и рейтинге
- 📋 Отображение всех фильмов в удобной таблице
- 🔍 Фильтрация по жанру и году выпуска
- 💾 Автоматическое сохранение данных в JSON
- 📂 Загрузка данных из JSON файла
- ✅ Валидация ввода (год - число, рейтинг от 0 до 10)
- 🗑️ Удаление выбранных фильмов
- 📊 Статус бар с информацией о количестве фильмов

## Требования

- Python 3.6 или выше
- Tkinter (обычно входит в стандартную установку Python)
- Операционная система: Windows, macOS или Linux
